## 概述

一个包含 21 款经典益智小游戏的网页合集，无需安装，打开即玩。所有游戏均使用原生 HTML/CSS/JS 开发，无任何第三方依赖。全面支持桌面键盘控制和移动端触屏操作。

## 技术实现

- 采用模块化游戏架构，每个游戏独立为一个 JS 模块
- 统一的游戏生命周期管理（初始化 / 开始 / 结束 / 销毁）
- 响应式布局，适配桌面和移动端
- 移动端触屏适配（滑动手势、屏幕方向键、触摸按钮）
- CSS 动画和过渡效果提升交互体验
